Run Info

It's important for others to know what setup you're using for your speedrun, in order to compare it to other, similar speedruns. Splitter makes it easy to include this information in your run.

Supported Fields

Title Game Title
Category Speedrunning category you're attempting to run
Attempt The number of times you have attempted to perform this run
Platform The platform the game is being played on (i.e N64, GameCube, Emulator, etc.)
Game Version Version number for the version of the game used in the run (i.e. "1.1.3"). Leave this blank if it doesn't apply to yoy
Game Region Region of the game used for the run (i.e. USA, JPN, EU)

Other timers may not recognize all these fields, particularly the Game Version field.

Setting Run Info

You can set any of fields listed above in the Info panel. In the Menu Bar, click Run -> Get Info, or click the button near the title.
